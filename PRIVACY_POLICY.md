# 隱私權政策（Privacy Policy）

**最後更新日期：2025-06-26**

感謝您使用「RecordTimeLabel」（以下簡稱「本擴充功能」或「本服務」）。本文件將說明我們如何蒐集、使用、儲存及保護您在使用本擴充功能時所提供或產生的資料。

RecordTimeLabel 是一個專為 YouTube 和 Twitch 平台設計的時間標記記錄工具，幫助用戶記錄直播和影片中的重要時間點。我們致力於保護您的隱私，並確保您的資料安全。

---

## 1. 我們蒐集哪些資料

### 1.1 本地儲存資料
本擴充功能主要將資料儲存在您的瀏覽器本地，包括：
- **時間標記記錄**：您在 YouTube/Twitch 上建立的時間戳、筆記內容
- **資料夾結構**：您建立的資料夾、記錄分類和排序
- **使用者偏好設定**：介面語言、主題模式、時間偏移設定等
- **UI 狀態**：資料夾展開狀態、面板大小等介面設定

### 1.2 雲端同步資料（僅在您選擇登入時）
當您選擇使用雲端同步功能時，我們會蒐集：

| 資料類型 | 說明 | 用途 |
| ---- | ---- | ---- |
| 身份驗證資料 | Google OAuth、Facebook OAuth 或 Email/密碼 | 用於身份驗證和帳戶識別 |
| 同步資料 | 您的時間標記、資料夾結構、設定等 | 在不同裝置間同步您的資料 |
| 使用統計 | 匿名化的錯誤日誌和基本使用資訊 | 改善服務品質和修復問題 |

### 1.3 我們不會蒐集的資料
- ❌ 瀏覽器歷史記錄
- ❌ 其他網站的 Cookies 或個人資料
- ❌ 您觀看的影片內容或觀看習慣
- ❌ 裝置資訊或位置資料
- ❌ 與本擴充功能無關的任何個人資訊

---

## 2. 資料如何被使用

### 2.1 核心功能
- **時間標記記錄**：在 YouTube 和 Twitch 頁面上記錄重要時間點
- **資料組織**：建立資料夾、分類和管理您的時間標記
- **多語言支援**：根據您的語言偏好顯示介面（支援繁體中文、英文、日文）
- **主題切換**：支援淺色和深色主題模式

### 2.2 雲端同步功能（可選）
- **跨裝置同步**：在不同瀏覽器和裝置間同步您的資料
- **資料備份**：將您的時間標記安全地儲存在雲端
- **智慧合併**：登入時自動偵測本地與雲端資料差異，提供合併選項
- **即時同步**：資料變更時自動上傳到雲端

### 2.3 資料處理原則
- **最小化原則**：僅處理提供服務所必需的資料
- **透明化**：所有資料處理都會明確告知用戶
- **用戶控制**：您可以隨時選擇停用雲端同步或刪除雲端資料
- **安全優先**：所有資料傳輸都使用加密連線

## 3. 第三方服務

本擴充功能使用以下第三方服務來提供雲端同步功能：

| 服務提供商 | 服務內容 | 用途 | 隱私權政策 |
| ---- | ---- | ---- | ---- |
| Google Firebase | Authentication & Cloud Firestore | 使用者身份驗證、雲端資料儲存與同步 | [Google 隱私權政策](https://policies.google.com/privacy) |
| Google OAuth | Chrome Identity API | Google 帳號登入驗證 | [Google 隱私權政策](https://policies.google.com/privacy) |
| Facebook OAuth | Chrome Identity API | Facebook 帳號登入驗證 | [Facebook 隱私權政策](https://www.facebook.com/about/privacy) |

### 3.1 資料處理說明
- **Firebase**：僅在您選擇雲端同步時使用，用於安全地儲存和同步您的時間標記資料
- **OAuth 服務**：僅用於身份驗證，我們不會存取您的社交媒體內容或聯絡人
- **本地優先**：即使不使用任何第三方服務，本擴充功能仍可完全正常運作

## 4. 資料儲存與安全措施

### 4.1 本地資料安全
- **瀏覽器儲存**：使用 Chrome Extension Storage API 安全地儲存在您的瀏覽器中
- **本地加密**：敏感設定資料在本地儲存時會進行適當的保護
- **隔離儲存**：資料僅限本擴充功能存取，其他網站或擴充功能無法讀取

### 4.2 雲端資料安全
- **傳輸加密**：所有資料透過 **HTTPS/TLS 1.3** 加密傳輸
- **儲存加密**：資料在 Google Cloud Firebase 中採用企業級加密儲存
- **存取控制**：嚴格的 Firebase Security Rules 確保只有您本人可以存取您的資料
- **身份驗證**：多重身份驗證機制，支援 OAuth 2.0 標準

### 4.3 密碼安全
- **不儲存明文**：密碼永遠不會以明文形式儲存
- **安全驗證**：Email/密碼登入透過安全的 Cloud Function 處理
- **權杖機制**：使用 Firebase Custom Token 進行安全的身份驗證

## 5. 資料分享與揭露

我們 **不會** 與任何第三方分享、出售或出租您的個人資訊；唯一例外是依法律要求或司法機關正式請求時，才可能依法配合提供。

---

## 6. 資料保留與刪除

### 6.1 本地資料控制
- **完全控制**：您可以隨時在擴充功能中刪除任何本地資料
- **瀏覽器清理**：清除瀏覽器資料時，本擴充功能的資料也會一併清除
- **選擇性刪除**：可以選擇性地刪除特定資料夾或時間標記

### 6.2 雲端資料刪除
您可以透過以下方式刪除雲端資料：
- **雲端同步狀態面板**：點擊「刪除雲端資料」按鈕
- **登出功能**：選擇登出並刪除雲端資料
- **重設合併模式**：清除同步設定並重新選擇資料處理方式

### 6.3 資料保留政策
- **即時刪除**：您要求刪除的資料會立即從我們的系統中移除
- **無備份保留**：刪除後的資料不會在我們的系統中保留任何備份
- **不可復原**：一旦確認刪除，資料將永久移除且無法復原

## 7. 您的權利

根據相關隱私法規，您享有以下權利：

### 7.1 資料存取權
- **查看資料**：隨時檢視您在擴充功能中儲存的所有內容
- **匯出資料**：使用內建的匯出功能下載您的所有資料
- **資料透明**：了解我們如何處理您的資料

### 7.2 資料控制權
- **修改權利**：隨時編輯、更新您的時間標記和設定
- **刪除權利**：隨時刪除部分或全部資料
- **撤回同意**：隨時停用雲端同步功能

### 7.3 資料可攜權
- **標準格式**：支援 JSON 格式的資料匯出
- **完整資料**：匯出包含所有時間標記、資料夾結構和設定
- **跨平台**：匯出的資料可以在其他裝置上重新匯入

### 8.1 變更通知
當本隱私權政策有重大變更時，我們會：
- 在擴充功能中顯示更新通知
- 在 GitHub 專案頁面發布變更說明

### 8.2 生效時間
- 政策變更將在發布後 30 天生效
- 繼續使用本擴充功能即表示您同意新的政策條款
- 如不同意變更，您可以選擇停止使用並刪除相關資料

## 9. 兒童隱私保護

本擴充功能不會故意蒐集 13 歲以下兒童的個人資訊。如果我們發現無意中蒐集了兒童的個人資訊，我們會立即刪除這些資訊。

## 10. 國際資料傳輸

您的資料可能會在不同國家/地區之間傳輸和處理，包括但不限於：

我們確保所有跨境資料傳輸都符合適用的隱私法規。
